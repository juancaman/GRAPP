## 🚀 DEPLOY EN 3 PASOS

### Paso 1: Setup (5 min)
```bash
# Copiar archivos
cp index_secure.html index.html
mkdir api && cp api_db_secure.js api/db.js
cp netlify_secure.toml netlify.toml
cp package_secure.json package.json

# Crear .env.local (llenar variables de Neon, Clerk, ImgBB)
cp .env.local.example.secure .env.local

# Instalar dependencias
npm install @neondatabase/serverless jose
```

### Paso 2: Database (1 min)
```
1. Ir a https://neon.tech dashboard
2. Crear proyecto
3. Ir a SQL Editor
4. Pegar contenido de NEON_SCHEMA_SECURE.sql
5. Click Execute
6. ✅ Base de datos lista
```

### Paso 3: Deploy (5 min)
```bash
netlify login
netlify deploy --prod
# Agregar env vars en Netlify Dashboard
# Redeploy
# ✅ Live!
```

**Total: 15 minutos**
