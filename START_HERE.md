## 🚀 EN 3 PASOS

### 1. Preparar
```bash

# Editar con tus claves (Neon, Clerk, ImgBB)
npm install
```

### 2. Testear
```bash
npm run dev

```

### 3. Deployar
```bash
netlify deploy --prod
```
